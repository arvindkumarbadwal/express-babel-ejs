# EJS Babel ExpressJs

Simple Example for Babel EJS implementation on ExpressJs

## How to use

### Run in Development Mode

```bash
npm run dev
```

### Run in Production Mode

```bash
npm run production
```

### Test

```bash
http://localhost:3000
http://localhost:3000/{you_name}
```
